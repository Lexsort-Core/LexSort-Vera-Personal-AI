Quick Installation Guides

🍏 macOS Setup

  1. Download the .dmg installer matching your chip architecture (Apple Silicon vs Intel).
  2. Open the DMG and drag LexSort Personal AI into your Applications folder.
  3. Because VERA runs as unsigned freeware, on first open you may need to right-click the application icon and select Open, then approve the prompt.

🪟 Windows Setup

  1. Download the Windows .zip installer.
  2. Extract the zip archive and run the executable installer (e.g. `.exe` or `.msi`).
  3. Accept the SmartScreen prompt (select More Info and click Run anyway) if displayed for this unsigned local app.

🐧 Linux Setup

  1. For Debian/Ubuntu, download the .deb package and install via: `sudo dpkg -i lexsort-personal-ai_0.1.0_amd64.deb`.
  2. Alternatively, download the standalone .AppImage package.
  3. Make the AppImage executable: `chmod +x *.AppImage` and run directly.
